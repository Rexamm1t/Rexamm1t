### Installation & Usage

```bash
# Quick install via Magisk/KernelSU
1. Download latest release
2. Install through module manager  
3. Reboot and configure
```

Configuration Options:

· NextRAM Manager App (GUI)
· Manual config: /data/adb/modules/nextram/config.conf
· Logs: /data/adb/modules/nextram/logs/

Technical Details

Compatibility:

· Android 7.0 and higher
· Magisk and KernelSU support
· Architectures: ARM, ARM64, x86, x86_64
· Root access required

Security:
Module operates at kernel level with minimal permissions.All changes are reversible and don't affect system integrity.
